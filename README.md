# Overview
This is a personal portfolio project, it showcases qualifications, familiar tech stacks,
previous work experience and notable projects. The website is responsive and adapts to
different screen sizes using CSS media queries. It also includes a navigation menu that
switches between desktop and mobile views.

# Structure
The project consists of the following:

- /assets
  - /images
  - resume.pdf
- /styles
  - main.css
- app.js
- index.html
- README.md
